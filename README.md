###  小美收银管理软件

* 前端基于react + ant design
* 后端基于nodejs koa web框架 + sequelize + mysql
* 已对接微信公众号／微信小程序／微信支付／支付宝支付／百度人脸抓拍
* 全栈开发 **In Free Time**
* [小美收银管理平台](https://www.ixiaomu.cn/xiaomei)
